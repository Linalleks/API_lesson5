
# Сравниваем вакансии программистов

Приложение анализирует вакансии разработчиков Москвы с HeadHunter и SuperJob по средней зарплате для каждого языка программирования из списка:
Python, Java, Javascript, Go, C++, C#, PHP, Ruby.

## Общие требования

1. Python3 должен быть уже установлен.
2. Затем используйте `pip` (или `pip3`, есть конфликт с Python2) для установки зависимостей:  
    ```  
    pip install -r requirements.txt  
    ```  
3. *Рекомендуется использовать [virtualenv/venv](https://docs.python.org/3/library/venv.html) для изоляции проекта.
4. Для работы приложения потребуется получить Secret key от приложения на SuperJob. Он нужен для взаимодействия с API SuperJob.  
Для этого не нужно писать код, достаточно зарегистрировать приложение на [SuperJob](https://api.superjob.ru/info/).  
Полезная информация:
    - заполняйте анкету для регистрации приложения минимально и просто. Главная цель - получить Sekret key. Остальные данные не повлияют на текущее приложение.
    - При регистрации приложения от вас потребуют указать сайт. Введите любой, это не проверяется. (например: https://example.com)
    - Secret key появится сразу после регистрации в параметрах доступа приложения. Выгядит он примерно так: `v2.f.1293456789.a4bc0f733fa28ca045f21acf233534ecba6aed12.f3cc882a8b1b2e7748c3e4af9c41e0d8185e5c99`.

5. Когда Secret key будет получен, в папке с текущим приложением создайте файл ".env".  
   В этом файле укажите токен в переменной `SJ_SECRET_KEY`.  
   Например,
    ```
    SJ_SECRET_KEY=v2.f.1293456789.a4bc0f733fa28ca045f21acf233534ecba6aed12.f3cc882a8b1b2e7748c3e4af9c41e0d8185e5c99
    ```

## Запуск приложения

Когда все необходимые требования для работы выполнены, через командную строку можно запускать приложение.  
Для этого потребуется команда:
```
python [относит_путь_до_файла_приложения]
```
Для проверки работы приложения можете воспользоваться подготовленным примером ниже.  
Чтобы запустить команду из примера без изменений, в командной строке перейдите в папку с приложением.

### Пример запуска приложения 

Введите в командной строке следующую команду и нажмите Enter
```
python .\main.py
```
